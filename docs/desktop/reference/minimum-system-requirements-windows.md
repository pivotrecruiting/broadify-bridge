# Broadify Bridge: Mindest-Systemanforderungen für Windows

Stand: 2026-04-20

## Unterstützte Systeme

- Windows 10 64-Bit
- Windows 11 64-Bit

## Mindestanforderungen

- Prozessor: aktueller 64-Bit-Prozessor mit mindestens 4 Kernen
- Arbeitsspeicher: mindestens 8 GB RAM
- Freier Speicherplatz: mindestens 1 GB
- Grafikeinheit: GPU mit aktueller Treiberunterstützung
- Bildschirm: externer Monitor, wenn Broadify Bridge für Video-/Grafikausgabe genutzt wird
- Netzwerk: stabile Netzwerkverbindung, idealerweise per LAN

## Empfohlen für produktiven Betrieb

- Prozessor: 6 Kerne oder mehr
- Arbeitsspeicher: 16 GB RAM
- Freier Speicherplatz: 2 GB oder mehr
- Laufwerk: SSD
- Grafikeinheit: dedizierte GPU oder moderne integrierte GPU
- Bildschirm: externer Monitor mit der tatsächlich benötigten Auflösung und Bildrate
- Netzwerk: kabelgebundene LAN-Verbindung

## Wichtige Hinweise für Endkunden

- Es werden nur 64-Bit-Windows-Systeme unterstützt.
- Für stabile Realtime-Grafikausgabe sind aktuelle Grafiktreiber wichtig.
- Bei Nutzung mit externer Video-/Grafikausgabe sollte kein älteres Office- oder Low-End-System verwendet werden.
- Für 4K-Ausgabe oder höhere Bildraten wird ein deutlich leistungsfähigeres System empfohlen.
- Virtuelle Maschinen oder stark eingeschränkte Remote-Umgebungen sind für produktive Grafik-Ausgabe nicht empfohlen.

## Zusätzliche Anforderungen je Einsatz

### Standardbetrieb

- Windows 10 oder Windows 11
- 8 GB RAM
- 1 GB freier Speicher
- stabile Netzwerkverbindung

### Produktiver Grafikbetrieb

- 16 GB RAM
- SSD
- externer Monitor
- aktuelle GPU-Treiber
- kabelgebundene Netzwerkverbindung

### 4K- oder performancekritische Nutzung

- 16 bis 32 GB RAM
- schneller Mehrkern-Prozessor
- starke GPU
- SSD
- externer 4K-fähiger Monitor

# Integration – Glossar

## Begriffe
- **OutputKey**: Kennung des Output‑Modus (`key_fill_sdi`, `video_sdi`, ...)
- **Layer**: Eine gerenderte Grafik‑Ebene
- **Preset**: Gruppe von Layern unter gleicher `presetId`
- **PortRole**: Rolle eines Ports (`fill`, `key`, `video`)
- **DeviceDescriptor**: Normiertes Geräte‑Objekt der Bridge
- **Graphics Bundle**: HTML/CSS + Defaults + Assets
- **Renderer**: Offscreen‑Rendererprozess für HTML/CSS
- **Helper**: Native DeckLink‑Binary für Playback/Device‑Listing
